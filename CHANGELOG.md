# Release history

## Version 0.1

First light!

### New features

* Bayesian optimization via `gp_minimize`. 
* Ensemble of and single decision tree based sequential model-based optimization via `forest_minimize` and `gbrt_minimize`.
* Convergence plots.
* Specifying and sampling from a parameter space.
